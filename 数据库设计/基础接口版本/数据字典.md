# 基础接口实现表数据字典
## user 表
```
属性                类型          长度       描述
user_id             int           64        用户id，唯一，标识用户
name                varchar       50        用户昵称
username            varchar       50        用户账号
password            varchar       50        用户密码
follow_count        int           32        用户关注总数
follower_count      int	          32        用户粉丝总数
```
## video 表
```
属性                类型          长度       描述
video_id            int           64         视频id，唯一，标识视频
play_url            varchar       2083       视频播放地址
cover_url           varcahr       2083       视频封面地址
favorite_count      int           32         视频总点赞数
comment_count       int           32         视频总评论数
```
## comment 表
```
属性                类型          长度       描述
comment_id          int           64         评论id，唯一，标识评论
user_id             int           64         用户id，唯一，标识用户
video_id            int           64         视频id，唯一，标识视频
content_place       varchar       255        评论内容存放的地址
create_date         datetime                 评论创建的时间
```

```
属性                类型          长度       描述
```
